Houston
=======

Центр управления полётами. Центр решения проблем. База знаний. Книга жалоб и предложений. :)

# Настройка (Development)
    git clone git://github.com/akolosov/houston.git
    cd houston
    bundle install
    rake db:migrate
    rake db:seed

# Использование (Development)
    rails s
    open http://0.0.0.0:3000/
